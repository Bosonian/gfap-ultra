# Usability and Use Specification

## Intended Users
Clinicians and researchers evaluating suspected stroke patients.

## Use Environment
- Embulance, Emergency room or lab
- Mobile or desktop browser interface.

## Key Functions
| Function | User Action | System Response |
|-----------|--------------|------------------|
| Data Entry | Fill patient details | Form validation and feedback |
| Risk Prediction | Submit form | Display ICH/LVO probabilities |
| Visualization | View results | Animated rings and gauges |
| Export | Click “Print Results” | Generate printable output |

## Potential Use Errors
| Error | Cause | Mitigation |
|--------|--------|-------------|
| Wrong score entered | Typo | Clear labels and validation |
| Misinterpreted color scale | Inexperience | Legend and text summary |
| App not loading offline | Cache not updated | PWA install prompt |

## Clinical Disclaimer
> This tool is for **research and educational use only**.  
> It does not replace professional medical judgment.
